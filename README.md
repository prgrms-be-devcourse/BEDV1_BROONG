## Git **Commit Message 전략**

- **추가, 생성, 수정, 변경, 제거, 삭제** 등 어떤 활동을 했는지 확실하게 작성하기

```java
⭐ feat : 새로운 기능 추가
⚙️ chore : 빌드 관련 업무 수정
🐞 fix : 버그 수정
📖 docs : 문서 수정
💅 style : 코드 스타일(포맷, 세미콜론 수정 등) 수정
♻️ refactor : 코드 리팩토링
🚦 test : 테스트 코드 수정 및 추가
🎉 init : 최초 커밋 
```
